自动打卡

原理：

  1. 1.post填表—网络编程技术
  2. 2.fiddler抓包技术
  3. 3.云函数—–定时执行函数(代码)

post

一个URL地址，它用于描述一个网络上的资源，而HTTP中的GET，POST，PUT，DELETE就对应着对这个资源的查，改，增，删4个操作。到这里，大家应该有个大概的了解了，GET一般用于获取/查询资源信息，而POST一般用于更新资源信息。

抓包

要实现对App的网络数据抓包，需要监控App与服务器交互之间的网络节点，监控其中任意一个网络节点（网卡），获取所有经过网卡中的数据，对这些数据按照网络协议进行解析，这就是抓包的基本原理。

实现原理

校园健康打卡 就相当于更新, 所以用post 发送
而post发送什么呢?
这里就需要我们抓包,看看抓取的包 它干了什么,然后我们模拟一个环境,发送post 就实现了校园健康打卡.

public static String Merge_string(String[] strings){
    String tr=null;
    for (int i = 0; i < strings.length; i++) {
        if(i!=strings.length-1){
            tr+=strings[i];
            tr+="=";
        }else {
            tr+=strings[i];
        }
    }
    return tr;
}

public static void post (String url,String header,String body)throws Exception{
    URL obj = new URL(url);
    HttpURLConnection con =(HttpURLConnection) obj.openConnection();
    //添加请求头
    con.setRequestMethod("POST");



    setHeader(con,header);
    //发送Post请求
    con.setDoOutput(true);
    DataOutputStream wr = new DataOutputStream(con.getOutputStream());
    wr.writeBytes(body);
    wr.flush();
    wr.close();
    int responseCode = con.getResponseCode();
    System.out.println("\nSending 'POST' request to URL : " + url);
    System.out.println("Post parameters : " + body);
    System.out.println("Response Code : " + responseCode);

    BufferedReader in = new BufferedReader(
            new InputStreamReader(con.getInputStream(),"utf-8"));
    String inputLine;
    StringBuffer response = new StringBuffer();



    while ((inputLine = in.readLine()) != null) {
        response.append(inputLine);
    }
    in.close();

    //打印结果
    System.out.println(response.toString());



}
